# Contribuir

Este repositório é uma fotografia de quatro padrões que usei no MBA. Não há promessa de manutenção activa, mas contribuições são bem-vindas.

## Como contribuir

1. **Issues** — para reportar erros, falhas factuais, ou sugestões. Diz claramente em que padrão (1/2/3/4) e o ficheiro afectado.
2. **Pull Requests** — para correcções pequenas (typos, links partidos, clarificações). Para mudanças grandes, abre primeiro uma issue para discutir o âmbito.

## Antes de submeter

- Verifica que o conteúdo continua **sanitizado** (sem dados pessoais, institucionais, IPs ou hostnames identificáveis)
- Mantém o estilo PT-PT (ou EN nos ficheiros já em EN)
- Para mudanças que toquem múltiplos padrões, divide em commits separados

## Licença das contribuições

Ao submeteres, concordas que a tua contribuição fica licenciada sob CC BY-NC-SA 4.0 (a mesma licença do repositório).

## Código de conduta

Sê construtivo. Diferenças de opinião sobre métodos são bem-vindas; ataques pessoais não.
